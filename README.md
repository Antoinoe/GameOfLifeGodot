# Game Of LIfe in Godot
## Technical Informations : 
- Godot Csharp 4.5.1

## How to install
- git clone
- open project

## How to use
- Go to the GOL.tscn
- Click on TileMapLayer
- To modify the tileset, go to Tileset (bottom of the screen), and paint where you want (inside the limit (for now 400*400))
- You can modify the script attached to it to either start with random cells, or keep your initial draw
